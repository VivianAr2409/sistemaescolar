# Sistema Escolar - VivianAr2409

Este es un sistema de gestión escolar desarrollado como proyecto independiente.

## Descripción
Sistema web para la gestión de datos escolares con funcionalidades de análisis estadístico y visualización de datos.

## Características
- Panel de control interactivo
- Análisis estadístico (histogramas, Pareto, estratificación)
- Gestión de usuarios con sistema de login
- Interfaz responsiva y moderna

## Tecnologías utilizadas
- Python/Django
- HTML5, CSS3, JavaScript
- Chart.js para visualizaciones

## Autor
Desarrollado por VivianAr2409

## Instalación
1. Clonar el repositorio
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar migraciones: `python manage.py migrate`
4. Iniciar servidor: `python manage.py runserver`
